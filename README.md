# Task Management Application (React)

A full-featured task management application built using React with API integration.

## Features
- Add, edit, and delete tasks
- Toggle task completion
- Task history by date
- Carry-over unfinished tasks
- Light/Dark theme support

## Tech Stack
- React(Vite)
- JavaScript (ES6+)
- CSS
- React Router
- REST API (json-server)
- Reducer pattern for state management
- Vercel (frontend deployment)
- Render (backend deployment)

## Project Structure
- Components-based architecture for scalable UI development
- Services layer abstraction for API integration
- Reducer pattern for centralized state management
